# ESP32 Temperature and Humidity Monitoring using DHT22

This project demonstrates how to monitor temperature and humidity using a DHT22 sensor with an ESP32 in the Wokwi simulator. The ESP32 reads sensor data, calculates the heat index, and displays the results on the Serial Monitor every 2 seconds.

## Components
- ESP32 Dev Board
- DHT22 Temperature and Humidity Sensor

## Working
- Reads humidity from the DHT22 sensor.
- Reads temperature in Celsius (°C).
- Reads temperature in Fahrenheit (°F).
- Calculates the Heat Index in both Celsius and Fahrenheit.
- Displays all readings on the Serial Monitor every 2 seconds.
- Prints an error message if the sensor fails to respond.
